# Architecture Docs

These documents explain how the example apps in this sandbox work under the hood. They are written for learning — if you are curious about how the code is organized and why, start here.

**You do not need to read these to use the sandbox.** They are here for when you want to go deeper.

## Where to Start

- **[TicTacToe Architecture](./tictactoe.md)** — A full walkthrough of the tic-tac-toe game (frontend + backend). This is the main reference implementation and the best place to start understanding how a real web app works end-to-end.
